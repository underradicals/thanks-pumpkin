A Stack is a Last-In-First-Out Data Structure. This means that elements are inserted from back to front (bottom to top), so that when it comes to removing elements, the last element that was inserted will be removed first. Elements are not just stored but are also implicitly ranked based on their insertion order, that means they have an Ordinality, namely the first element in is the last to be removed, and the last element in will be the first to be removed. The most recently added element has the highest "rank" or is the first in line to be accessed or removed.
